////////////Установка менеджера////////////////

1) Установить движок (Johncms) менеджера, перейти по ссылке installjo/
2) Установить базы менеджера , перйти по ссылке installman/ins/
3) Сделать запрос в базу через PHPMyAdmin:

INSERT INTO `cms_settings` (`key`, `val`) VALUES ('mod_manager', '1')

4) В принцыпе всё) Если возникнут ищё вопросы, пишите на warriorworld@bk.ru
